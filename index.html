Helper to translate shell commands to Ansible playbook entries.

    Samples (click on a sample to try it)

    • apt-get -y install toilet
    • apt-get remove lolcat
    • sudo add-apt-repository -y ppa:foo/bar
    • pip install project_runpy==0.3.1
    • npm install -g grunt-cli

    Limitations

    Very few commands are supported. Basically, just what's in the samples.

    Options must be specified like --port=8000, not --port 8000
